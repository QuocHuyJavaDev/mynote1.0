About MyNote 1.0

Notes taking app

  • Take notes, create todo-list and schedule events
  • Developed by .Net MAUI framework
  • Support platforms: Android, IOS, macOS, Windows, Linux
  • Expected release date: March 3, 2023

Key functions

Taking notes

Add text, images, audio, documents, code, video. Organize notes in a '1 notebook - many notes' model. Automatically import content into notes. Export as PDF files. Share via email.

Create Todo-lists

Create a list of tasks (things to do). Mark tasks as completed or not. Support reminder for tasks by automatically transferring tasks that need to be reminded to the scheduling function.

Schedule management

Create a schedule with customizable time. Set up reminders with alarm sound. Trigger alarm and send email notification when the time comes. Repeat the task on the desired schedule
